大阪釣りスポットは、大阪の釣り場を紹介し、お気に入り登録で保存したり釣果などをコメントできるアプリです。「大阪の釣り場ってあるのかな？」、「どんな魚が釣れるのかな？」という方におすすめです。

制作背景
より多くの人に大阪の釣り場を知ってほしいと思いこのアプリを作成しました。


使用技術
* HTML / CSS
* Javascript
* JQuery
* Bootstrap
* Ruby 2.6.3
* Rails 6.0.3.4
* MySQL
* AWS(EC2, VPC, Route53, S3)
* Doker/docker-compose
* CircleCI(CI/CD)
* RSpec

機能一覧
ユーザー機能
* ユーザーの新規登録、プロフィール画像の追加、編集
* ゲストログイン
googlemap機能
* 釣り場をgooglemapで表示
天気予報機能
* 釣りで重要な天気予報を表示
お気に入り機能
* 釣り場のお気に入り登録、お気に入り解除
検索機能
* 釣り場の検索
ランキング機能
* 釣り場のお気に入り数のランキング
コメント機能
* 釣り場コメントの新規投稿、削除
その他
* Rspecによるテスト
